# 🌐 MTProxy Dynamic Relay

Динамический TCP-релей для Telegram MTProxy с веб-админкой, мониторингом RTT и автоматическим выбором лучшего прокси.

## ✨ Возможности
- 📊 **Веб-админка**: создание списков, добавление прокси, мониторинг в реальном времени
- ⚡ **Динамический релей**: автоматическое открытие портов, маршрутизация на прокси с минимальным RTT
- 🔍 **Health-check**: проверка TCP-доступности каждые 15 сек, фильтрация "мёртвых" нод
- 💾 **SQLite WAL**: лёгкая БД, высокая отказоустойчивость, нулевые зависимости
- 🔒 **Безопасность**: авторизация, изоляция портов, graceful shutdown
